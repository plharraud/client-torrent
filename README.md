# BITTORRENT

# Build
`gradle build`

# Usage
`gradle run`

`java -jar app/builds/libs/client.jar -<debug|info> <torrent> <destination>`

log verbosity level : debug > info > error (default)
[etc.](https://logging.apache.org/log4j/2.x/log4j-api/apidocs/index.html)
